<h2 align="center">How it Works</h2>

<p align="center">
  The majority of this add-on is based upon existing machines and recipes from Create.<br/>
  Create uses Encased Fans with specific block types and fluids to create processes using a Fan.<br/>
  Pressing and Mixing processes allow for further progression while maintaining Create's base loop.<br/><br/>
  JEI will show available categories when they are enabled.<br/>
  The configuration options allow server owners and pack makers to turn processing types on and off.
</p>

<h2 align="center">Config File</h2>

<p align="center">
  The configuration file allows for enabling and disabling processing families.<br/>
  This makes it easy to create a pack with only a few mechanics, or a more focused setup on a server.
</p>

<h2 align="center">Additional Comments</h2>

<p align="center">
  The vision for this add-on is still to feel like Create, just with more options for processing items and a few extra machine types for existing machines.
</p>
